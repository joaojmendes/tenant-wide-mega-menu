# Support policy

This repository supports public testing of Tenant Wide Mega Menu.

## Supported requests

- Installation and tenant-wide activation problems.
- Configuration and manager-authorization problems.
- Reproducible UI, navigation, audience, or tool defects.
- Compatibility reports for supported SharePoint Online experiences.
- Documentation corrections and focused feature requests.

## Before opening an issue

1. Read [Installation](docs/INSTALLATION.md), [Configuration](docs/CONFIGURATION.md), and [Troubleshooting](docs/TROUBLESHOOTING.md).
2. Search existing issues.
3. Reproduce with the latest release when possible.
4. Remove confidential information from logs and screenshots.

## Response expectations

This is a test and feedback channel, not a guaranteed service-level agreement. Reports are prioritized by reproducibility, severity, security impact, and available maintainer capacity.

GitHub Issues are public. Do not post access tokens, tenant secrets, license tokens, personal data, private URLs, or customer content.
